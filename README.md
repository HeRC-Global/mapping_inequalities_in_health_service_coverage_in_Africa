# Mapping inequalities in health service coverage in Africa 
## Introduction
This study examines disparities in access to essential health services across African countries and regions. While Africa has made strides in expanding health services, significant gaps remain, particularly in rural and impoverished areas. Achieving Universal Health Coverage (UHC) necessitates identifying and addressing these service gaps.
This is just a scope review, for more detailed information, please refer to the full report [here](Report.md)
